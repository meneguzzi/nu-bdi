Library Requirements
- Jason 1.3.1 (found at https://sourceforge.net/projects/jason/files/jason/version%201.3.1/)
- JasonEnv, a library of extended environments for Jason, used in the tests of nu-BDI
- JasonCSP, a tiny constraint satisfaction engine implemented using Jason's logic library
